    Chapter 4: Managing Product Variants & Add-ons#

    Pre-requisite: Before you can add variants (sizes/colors) or add-ons to a product, you must first define them in their respective sections in your dashboard menu.

    Step 1: Create Attributes & Add-ons First#

    • For Variants (e.g. Size, Weight, Color):
      1. Go to the dashboard menu and select Attributes (or Product Settings > Attributes).
      2. Add new attributes like Size, Weight, or Color.
    • For Add-ons (e.g. Extra Cheese, Extra Pav, Cold Drink combo):
      1. Go to the dashboard menu and select Add-ons.
      2. Click Add New Add-on, enter its name (e.g., Extra Cheese), set its price, and click Save.

    Step 2: Link them to your Product#

    1. When adding or editing a product, scroll down to the Attributes & Add-ons section.
    2. Select the checkbox for the attributes you want to use (e.g., Size).
    3. Choose the values (e.g., Small, Medium, Large).
    4. Set the specific price and stock for each combination under Variant Combinations.
    5. Select the Add-ons you want to offer with this product from the dropdown.
    6. Click Save.
